Panels Variant Page Title allows you to set your page title directly from Panels configuration, and also allows you to take advantage from Panels Arguments and Contexts as Placeholder tokens.

Panels Variant Page Title configuration saved with the Panels configuration so it is exportable with Features module.

Instalation
1. Unpacking it to your Drupal sites/all/modules directory if you're isntalling by hand.
2. Enable Panels Variant Page Title in Admin menu > Site building > Modules.
